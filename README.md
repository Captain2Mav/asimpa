# Site web ASIMPA

## Présentation du projet

Ce projet a été réalisé dans le cadre de ma formation Développeur Web et Web Mobile.

Il s’agit de la refonte et du développement du site web ASIMPA, un service social indépendant. 
L’objectif du projet est de rendre le site plus clair, plus lisible et plus accessible pour les différents publics : particuliers, entreprises, associations et organismes.

## Objectifs

- Présenter l’activité d’ASIMPA
- Améliorer la lisibilité des informations
- Organiser les contenus par type de public
- Faciliter la prise de contact
- Proposer une interface plus claire et plus professionnelle

## Technologies utilisées

- Symfony
- Twig
- HTML
- CSS
- Git / GitHub

## Fonctionnalités

- Page d’accueil
- Présentation du service
- Pages de prestations par public
- Page tarifs
- Page contact
- Formulaire de contact
- Page de connexion
- Forum / espace d’échange
- Mentions légales
- Politique de confidentialité
